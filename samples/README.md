# Sample Datasets

These datasets are intentionally small and synthetic. They are provided for local smoke checks and benchmark script validation.

## Included Datasets

- `datasets/renpy-mini`: RenPy-like folder layout with script, image, audio, and text assets.
- `datasets/unity-mini`: Unity-like folder layout with common marker files and representative binaries.
- `datasets/mixed-mini`: Generic mixed-assets layout used to validate fallback extraction and triage behavior.

## Notes

- Files are placeholders for workflow testing; they are not full game data.
- You can add your own folders under `samples/datasets` and re-run the benchmark script.
